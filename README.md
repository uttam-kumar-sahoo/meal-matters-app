## Features

- Easy to use and simple UI.
- Efficient data with MySql.
- User authentication.
- Admin dashboard.
- Feedback form with firebase.
- Map integration with Googlemap.


## Techstack

- Html, Css, Javascript
- Php, Mysql
- Firebase (firestore, authentication, storage, analytics)
- Googlemap API
- Gpay API for payment
